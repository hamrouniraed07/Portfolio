# Portfolio

## Project Description
This is a React-based personal portfolio website that showcases various sections including Home, About, Projects, Experience, Education, Interests, Certifications, and Resume. The portfolio utilizes React Router for seamless navigation between pages. 

The project includes user-friendly features such as a Preloader splash screen, a responsive Navbar, smooth scroll to top on navigation, and a Footer component with contact information.

## Technologies Used
- React
- React Router DOM
- Bootstrap for styling
- CSS (App.css, style.css)

## Project Structure
- `src/components/`: Contains reusable components and individual sections of the portfolio
- `src/App.js`: Main application entry point with routing setup
- `public/`: Static assets and the main HTML file (`index.html`)

## Installation & Running
To run this project locally:

1. Ensure you have Node.js and npm installed.
2. Clone the repository.
3. Navigate to the project directory in your terminal.
4. Run the following commands:

```bash
npm install
npm start
```

5. Open your browser and access the app at `http://localhost:3000`

## Features
- Multi-page navigation using React Router
- Responsive and mobile-friendly design
- Preloader splash screen for better user experience
- Smooth scroll to top when navigating to new pages
- Footer with contact details and important links
